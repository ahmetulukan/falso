# Ball Bounce Blitz

A fast-paced arcade brick breaker game built with Flutter and Flame engine.

## Features
- Classic brick breaker mechanics
- Enemy obstacles to avoid
- Score multiplier system
- Progressive difficulty levels
- Touch/drag controls

## Getting Started

```bash
flutter pub get
flutter run
```

## Game Controls
- Drag paddle left/right
- Tap to launch ball
- Break bricks, avoid enemies

## Architecture
- `lib/game/` - Main game class
- `lib/components/` - Ball, Paddle, Brick, Enemy
- `lib/systems/` - ScoreManager, LevelManager
